# EcoPrint Studio

> Soluciones industriales de impresión y tecnología de vanguardia para procesos de producción complejos. Calidad, precisión y eficiencia en cada detalle.

## 🚀 Acerca del Proyecto

**EcoPrint Studio** es una Single Page Application (SPA) moderna y responsiva, diseñada con una estética industrial-tech. El proyecto presenta un catálogo dinámico de servicios, secciones informativas detalladas, un sistema de cotizaciones y una navegación fluida optimizada para el rendimiento y la experiencia de usuario.

## 🛠️ Tecnologías y Stack

* **Core:** React, Vite
* **Estilos:** Tailwind CSS v4
* **Tipografía:** Montserrat (UI) y JetBrains Mono (Elementos técnicos)
* **Control de Versiones:** Git / GitHub
* **Despliegue:** Vercel
